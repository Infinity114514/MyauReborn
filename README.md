# MyauReborn

![Preview](/images/image2.png)

[Myau Client](https://myau.sell.app/), for those who can’t afford it.

Based on **`OpenMyau`**, with added features and improvements not found in the original, focused on expanding and refining the original Myau client.

## Changes

* Added Click GUI (ported from Raven B3 by blowsy)
* Fixed bugs
* Improved Scaffold and Autoblock
* Improved Velocity
* And more.

## Building

```bash
./gradlew build
```
